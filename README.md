### 🔥 Featured Projects

#### 📖 [QuranCompanion](https://github.com/FAIZAN-Bor/QuranCompanion)
*Our Capstone Final Year Project (FYP)* — A modern interactive Quran learning platform with integrated AI.
- **AI Recitation Analysis**: Speech-to-text pipeline utilizing fine-tuned Whisper (`tarteel-ai`) and Wav2Vec2 models for real-time pronunciation scoring.
- **Gamified Learning**: Built modules for Quran recitation, interactive Norani Qaida, daily Duas, and customizable quiz challenges with rewards and badges.
- **Tech Stack**: React, Node.js, Express, MongoDB, Python, Socket.IO, TailwindCSS.

#### 📊 [Biz Smart Tracker](https://github.com/saif55045/Biz-Smart-Tracker-System)
*All-in-One ERP Platform for Small & Medium Enterprises (SMEs).*
- **Multi-Tenant Dashboard**: Sales tracking, live inventory levels with custom fields, employee check-in/attendance, real-time team discussion boards, and structured reports.
- **High Security**: Integrated Helmet security headers, rate-limiting, and MongoDB query sanitization to prevent NoSQL injections.
- **Tech Stack**: React 19, Node.js, Express, Mongoose, Socket.IO, Passport.js OAuth.

#### 🎙️ [n8n Voice Cloning Workflow](https://github.com/saif55045/n8n_workflow_openvoice)
*Automated media processing pipeline integrating FastAPI and n8n.*
- **OpenVoice Pipeline**: Seamless orchestration of audio inputs through OpenVoice models to generate instant high-fidelity cloned vocal samples.
- **Tech Stack**: FastAPI, PyTorch, n8n Automation Engine, Docker, Bash.

#### 🤖 [Medical RAG & Compliance Checker](https://github.com/saif55045/Medicalbot_and_compliancechecker_RAG)
*LLM-based Retrieval-Augmented Generation (RAG) system for healthcare query processing and HIPAA compliance checks.*
- **Intelligent QA**: Resolves complex medical queries against customized local knowledge stores.
- **Tech Stack**: Python, LangChain, FAISS, OpenAI API, Streamlit.
